Navigation Directive

BossyUI's Navigation directive is designed to make it easy to put a navigation bar on your webpage.
Our menus have an identification system which allows you to pre-select a defaul menu to be open on page startup.
By hiding extraneous submenus from view as you click, we take the clutter out of deep menu navigation.

Usage

Creating the navigation menu is as easy as creating any standard html tag:

<bossy-navigation config="myConfigObj"></bossy-navigation>

Options

  • config - Object that specifies navbar configuration. One of the following are required to build a navigation bar:
  • config.menuObj - Object that specifies menu data/layout, as per the following example.

    Note: User need not specify an id for each menu, as our directive will generate unique menu id's if this attribute is unspecified. 

    {
    activeMenuId: 'menu1', //activeMenuId allows you to use a reference to a submenu to pre-select which menu you would like to be open.
    navigation: [
        {
            title: 'Menu1',
            id: 'menu1',
            subMenus: [
                {
                    title: 'Nested1',
                    id: 'nested1',
                    subMenus: [
                        {
                            title: 'Nested2',
                            url: 'nested2.com'
                        }
                    ]
                }
            ]
        },
        {
            title: 'Menu2',
            url: 'menu2.com'
        }
    ]
}
  • config.menuUrl - URL of API endpoint containing a menuObj, configured as the example above.
    This flexibility in design allows you to either specify a menu object in-line, or clean it up by putting it in a .json file, or even to get the object from a remote source via url.

2. In this example, we take a very large menu and show how our navigation bar cleans up its navigation by restricting the amount of screen real-estate that the menu occupies.


With the standard Tiger Direct menu, the screen is very quickly cluttered with the various submenus which belong to the navigation bar.

tiger-direct.json


tdMenuConfig = {
    menuUrl: 'tiger-direct.json',
}

<bossy-navigation config="tdMenuConfig"></bossy-navigation>
